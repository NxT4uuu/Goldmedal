# 🏅 GoldMedal Web Application

A full-stack web application that allows users to explore Olympic medal statistics by country.  
Built with **HTML, CSS, JavaScript (frontend)** and **Java, Spring Boot (backend)**, this project provides interactive features to search, sort, and analyze medal data.

---

## ✨ Features

- **Country Search**
  - Search for any country and view its medal details.
  - Display country summary including gold, silver, and bronze counts.

- **Sorting**
  - Sort countries by total medals (most → least).
  - Filter results by gender (male/female).
  - Filter results by season (summer/winter).

- **Detailed Views**
  - Country-specific medal breakdown.
  - Separate statistics for male vs. female athletes.
  - Separate statistics for summer vs. winter games.

- **Interactive Frontend**
  - Responsive UI built with HTML, CSS, and JavaScript.
  - Dynamic rendering of medal tables and charts.

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: Java, Spring Boot  
- **Database**: H2 (with `data.sql` for sample data)  
- **Build Tool**: Gradle  
- **Persistence**: Spring Data JPA  
- **API Style**: RESTful endpoints
